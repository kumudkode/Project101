HealthHub — Simple Health Management Demo

This folder contains a small, aesthetic single-page Health Management System demo inspired by the provided DevLab portfolio template.

Files:

- `index.html` — main UI (dashboard, patients, appointments, prescriptions, messages)
- `style.css` — styles, responsive layout
- `script.js` — demo data and interactivity (add patient modal, schedule appointment, tables)

Open `index.html` in a browser to view the UI.
