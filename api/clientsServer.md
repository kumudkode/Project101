# Client
- Any device that connects to the internet to get data from somewhere(makes a "request")
- client can be things like phone, laptop, smart devices.

# Server
- A server is a powerful computer that stores data and serves it to clients upon request.
- Basically just a computer. Accepts requests from client asking for something, then responds to the client with that thing
- e.g an HTML page, an image or file, or just plain data 
- Servers host APIs that clients can access to retrieve or manipulate data.

1) What are some examples of "clients" you've used today?
* Laptop
* Smart Phone
* Robot vacuum
* Smart plugs

2) How would you explain what a "server" is to a 5 year-old?
A computer that send my own computer things when I ask it to.

3) In what way do clients and servers interact with each other?
Client sends a request to a server, and the server sends back a response.
