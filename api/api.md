# API (Application Programming Interface)
- What is API?
-  In simpler term API is any tool that helps connect your program to another's program
-  API is a set of rules that allows different software entities to communicate with each other.
- Getting Data from a server
  - The server hosts "an API" -exposed "endpoints" we can access for getting data from the server.
  - Note that the server doesn't give us access to everything just the things they want us to have
  - We can send requests to the server's API endpoints to retrieve or manipulate data.

- Importance of API
   - Facilitates integration between different software systems
   - Enables automation of tasks
   - Enhances functionality by allowing third-party services
   - Promotes scalability and flexibility in software development
- Types of API
   - REST API
   - SOAP API
   - GraphQL API
   - Webhooks
